# TaskFlow Success Metrics (Reference)

## Adoption metrics

| Metric | Target (90 days) | How measured |
|--------|------------------|--------------|
| Active teams using TaskFlow | 6 product + 1 platform team | Weekly unique API users / UI logins |
| Platform tasks tracked | 100% of P0/P1 initiatives | Count of tasks with owner ≠ unassigned |
| Spreadsheet coordination | 0 platform spreadsheets | Manager self-report + audit |

## Delivery metrics (DORA-aligned)

| Metric | Baseline | Target | How TaskFlow helps |
|--------|----------|--------|-------------------|
| Deployment frequency | 6-week train | Weekly (pilot) | Visible pipeline tasks with done criteria tied to deploy |
| Lead time for changes | 3–5 days hotfix | < 48h pilot | Dependencies surfaced before merge |
| Time to restore | 4–14 hours | < 4 hours | Incident follow-up tasks owned and prioritized |
