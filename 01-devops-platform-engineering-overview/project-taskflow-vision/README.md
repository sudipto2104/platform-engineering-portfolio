# Project: TaskFlow Vision

Apply DevOps culture principles to define the vision for **TaskFlow** — the task management platform you will build across all 21 weeks of the bootcamp.

This is not just planning. It is practice thinking like a platform engineer who balances technical design with business value.

## Deliverables

Complete or refine these documents:

| File | Purpose |
|------|---------|
| `VISION.md` | Platform vision, users, value proposition |
| `FEATURES.md` | Core features with business justification |
| `USER_STORIES.md` | User stories from a DevOps / platform perspective |
| `SUCCESS_METRICS.md` | Metrics to measure platform value |

Templates are in `templates/`. Reference examples are in `examples/` — write your own before peeking.
