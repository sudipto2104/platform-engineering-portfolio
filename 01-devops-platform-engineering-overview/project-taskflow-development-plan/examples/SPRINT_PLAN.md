## Sprint S0 detail (Week 1 — complete)

- **Goal:** Runnable TaskFlow + platform thinking artifacts
- **Stories:** Vision, architecture, documentation, automation lab, transformation capstone
- **DoD:** All Week 1 check scripts pass

## Sprint S9 detail (MVP checkpoint)

- **Goal:** TaskFlow in cloud staging with CI/CD
- **Stories:** Cloud account, Terraform apply, smoke tests in pipeline
- **DoD:** MVP_DEFINITION criteria met

## Sprint S20 detail (Release)

- **Goal:** Production-ready portfolio showcase
- **Stories:** Security review, load test, runbooks, demo video script
- **DoD:** Week 21 hardening checklist 100%

## Velocity assumptions

- 1 engineer (student) @ 12–15 hours/week
- Carry-over allowed; MVP scope adjustable with mentor

## TaskFlow linkage

Each sprint creates TaskFlow tasks mirroring bootcamp work — dogfooding the product.
