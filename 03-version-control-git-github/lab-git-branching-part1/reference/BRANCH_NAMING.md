# TaskFlow Branch Naming

| Pattern | Example | Use |
|---------|---------|-----|
| `feature/<issue>-<slug>` | `feature/12-task-filters` | New capability |
| `fix/<issue>-<slug>` | `fix/45-health-json` | Bug fix |
| `chore/<slug>` | `chore/ci-ruff` | Tooling, deps |
| `docs/<slug>` | `docs/api-week3` | Documentation |

**Never** commit directly to `main` after Week 3 — use PRs.
